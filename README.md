# # SQL JOINs and Window Functions Project

## Business Problem
The company needs to analyze sales data to identify top products, customer behavior, and sales trends.

## Database Schema
The system includes customers, products, and sales tables linked using foreign keys.

## JOIN Queries
INNER, LEFT, RIGHT, FULL OUTER, and SELF JOINs were implemented to analyze relationships.

## Window Functions
Ranking, aggregate, navigation, and distribution window functions were used.

## Key Insights
Top products generate most revenue.
Customer spending varies significantly by region.
